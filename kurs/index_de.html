01

Was ist der Learning Skill?

Stellen Sie sich vor, jemand liest Ihren kompletten Code und baut daraus einen verständlichen Kurs — genau das macht der Learning Skill. Es ist ein Plugin für Claude Code, das zwei Superkräfte hat:

📚

Interaktive Kurse

Mehrstufige HTML-Kurse, die Ihre Codebase Schritt für Schritt erklären — von der Übersicht bis zum Detail.

🕸️

Knowledge Graphs

Interaktive Wissenslandkarten, die Zusammenhänge zwischen Komponenten sichtbar machen.

Beide Formate werden als selbstständige HTML-Dateien erzeugt — keine Installation, kein Server, einfach im Browser öffnen.

02

Die drei Modi

Je nachdem, was Sie brauchen, wählen Sie einen von drei Modi. Jeder Modus erzeugt andere Ausgabeformate — alle als sofort nutzbare HTML-Dateien.

🎓

Kurs-Modus

Erzeugt mehrstufige HTML-Kurse mit Modulen, Animationen und Navigation.

Details ▼

Ideal zum Onboarding neuer Teammitglieder oder zum Dokumentieren komplexer Projekte. Erzeugt bis zu 4 Ebenen: Überblick (L0), Vertiefung (L1), Details (L2) und Referenz (L3).

🧠

Verstehen-Modus

Erzeugt einen Knowledge Graph mit Dashboard und interaktiver Visualisierung.

Details ▼

Perfekt, um die Architektur einer Codebase auf einen Blick zu erfassen. Enthält Layers, geführte Tours und Filter — alles in einer einzigen HTML-Datei.

🔄

Kombiniert

Beides zusammen: Strukturierte Kurse plus Knowledge Graph in einem Durchlauf.

Details ▼

Der umfassendste Modus. Sie erhalten sowohl die linearen Kurse zum Durcharbeiten als auch den Graph zum Erkunden. Empfohlen für größere Projekte.

03

Einen Kurs erstellen lassen

In vier einfachen Schritten kommen Sie vom Code zum fertigen Kurs. Der Learning Skill führt Sie durch den Prozess — Sie müssen nur antworten.

1

Skill aufrufen

Tippen Sie /learning in Claude Code. Der Skill startet und analysiert Ihr Projekt.

2

Zielgruppe wählen

Wer soll den Kurs lesen? Anwender, Entwickler oder Stakeholder — der Inhalt wird darauf zugeschnitten.

3

Modus wählen

Kurs, Knowledge Graph oder beides? Sie entscheiden, welches Format Sie brauchen.

4

Fertige HTML-Dateien erhalten

Der Skill erzeugt alle Dateien automatisch in einem /kurs-Ordner. Einfach im Browser öffnen.

Mehr erfahren Schritt-für-Schritt-Anleitung → Alle Optionen, Parameter, Beispiel-Sitzung
L1
04

Den Knowledge Graph nutzen

Der Knowledge Graph zeigt Ihre Codebase als interaktive Wissenslandkarte. Komponenten werden zu Knoten, Abhängigkeiten zu Verbindungen — alles in einem Dashboard, das Sie erkunden können.

Graph-Ansicht

App.tsx Router AuthProvider API Client Store Dashboard

Knoten anklicken, um Details zu sehen. Zoomen und Verschieben wie in einer Karte.

Steuerung

Layers: Blendet Gruppen ein/aus — z.B. nur Frontend oder nur API.

Tours: Geführte Pfade durch die Architektur, Schritt für Schritt erklärt.

Suche: Finden Sie jede Komponente sofort per Texteingabe.

Tipps & häufige Fragen

Die wichtigsten Fragen zum Learning Skill — kurz und verständlich beantwortet.

Der Learning Skill funktioniert mit Projekten jeder Größe — von einzelnen Scripts bis zu Monorepos mit hunderten Dateien. Bei sehr großen Projekten werden automatisch die wichtigsten Bereiche priorisiert.
Alle, die Claude Code versteht — also praktisch jede gängige Sprache: JavaScript, TypeScript, Python, Java, Go, Rust, C#, Ruby und viele mehr. Auch Konfigurationsdateien und Infrastruktur-Code werden berücksichtigt.
Ja. Alle Ausgaben sind selbstständige HTML-Dateien ohne externe Abhängigkeiten (außer Google Fonts). Sie können sie bearbeiten, in Ihr Intranet stellen oder als Dokumentation ins Repository legen.
