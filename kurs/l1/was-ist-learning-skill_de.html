Was ist der Learning Skill?

Ein Claude-Code-Plugin, das beliebige Codebases in verständliche Lernmaterialien verwandelt. Für alle, die Code verstehen wollen — ohne den Code lesen zu müssen.

01

Überblick & Vision

Der Learning Skill ist ein Plugin für Claude CodeClaude Code ist Anthropics offizielle Kommandozeile für KI-gestützte Softwareentwicklung. Der Learning Skill erweitert Claude Code um Lernfunktionen., das eine grundlegende Herausforderung der Softwareentwicklung löst: Codebases sind schwer zu verstehen — besonders für Menschen, die nicht selbst am Code gearbeitet haben.

Statt Dokumentation manuell zu schreiben, analysiert der Learning Skill den gesamten Quellcode und erzeugt daraus automatisch zwei Arten von Lernmaterial:

📚

Interaktive HTML-Kurse

Strukturierte, mehrstufige Lernpfade — vom groben Überblick bis ins letzte Detail. Komplett im Browser nutzbar, ohne Installation.

🕸️

Knowledge Graphs

Visuelle Wissenslandkarten, die Zusammenhänge zwischen Komponenten, Modulen und Konzepten als interaktives Netzwerk darstellen.

Die Vision: Jeder Mensch soll jede Codebase verstehen können — unabhängig von technischem Vorwissen. Der Learning Skill ist der erste Schritt dorthin.

02

Für wen?

Der Learning Skill erzeugt Inhalte für drei verschiedene Zielgruppen. Jede bekommt genau die Perspektive, die sie braucht:

👤

Anwender

Nutzen die Software, kennen keinen Code. Bekommen erklärt, was die Software tut und wie sie funktioniert — ohne technischen Jargon.

💻

Entwickler

Arbeiten mit dem Code. Bekommen Architektur, Patterns, Datenflüsse und Implementierungsdetails — technisch fundiert.

📊

Entscheider

Verantworten das Projekt. Bekommen Risiken, Abhängigkeiten, Technologie-Stack und strategische Bewertungen.

Die Zielgruppen können kombiniert werden. Ein Kurs kann gleichzeitig Anwender- und Entwickler-Perspektiven enthalten — der Learning Skill segmentiert die Inhalte automatisch.

🧩 Kurz-Check: Welche Perspektive passt?

Sie sollen einem neuen Teammitglied erklären, wie die API-Endpunkte strukturiert sind. Welche Zielgruppe wählen Sie?

Richtig! API-Endpunkte sind ein technisches Thema. Die Entwickler-Perspektive liefert genau die Details zu Routen, Payloads und Datenflüssen.
Nicht ganz. API-Endpunkte sind ein technisches Detail — hier ist die Entwickler-Perspektive die richtige Wahl.
03

Was kommt raus?

Der Learning Skill erzeugt drei Arten von Ausgabeformaten. Alle sind self-contained HTMLJede HTML-Datei enthält alles, was sie braucht: CSS, JavaScript, Inhalte. Kein Server, keine Installation — einfach im Browser öffnen.-Dateien, die direkt im Browser funktionieren:

  • HTML-Kurse (L0 bis L3)
    Vier Tiefenebenen: L0 gibt den Überblick, L1 vertieft Themen, L2 erklärt Details, L3 ist technische Referenz. Jede Ebene verlinkt zur nächsten.
  • Knowledge Graphs mit Dashboard
    Interaktive Visualisierung aller Komponenten und ihrer Beziehungen. Mit Filtern, Suche, Layern und geführten Tours.
  • Kombinierte Ausgabe
    Kurs und Knowledge Graph zusammen. Der Graph liefert dabei ein Helpfulness-Scoring, das die Kursqualität verbessert.

Die Dateien können per E-Mail verschickt, auf einem Fileserver abgelegt oder in ein Wiki eingebettet werden. Keine Abhängigkeiten, keine Laufzeit-Anforderungen.

04

Einsatzgebiete

Der Learning Skill ist überall nützlich, wo Menschen Codebases verstehen müssen. Hier die häufigsten Szenarien:

🤝

Onboarding

Neue Teammitglieder verstehen das Projekt in Stunden statt Wochen. Kurse erklären Architektur, Konventionen und Zusammenhänge.

🔍

Code-Reviews

Reviewer bekommen den Kontext, den sie brauchen. Der Knowledge Graph zeigt, welche Teile zusammenhängen und was ein Change beeinflusst.

📝

Dokumentation

Statt manuell Docs zu schreiben, generiert der Skill aktuelle, interaktive Dokumentation direkt aus dem Code.

🎓

Lernen & Lehren

Open-Source-Projekte, Tutorials, Uni-Kurse — der Skill macht jeden Code zum Lehrmaterial mit anpassbarer Tiefe.

Ob kleines Nebenprojekt oder Enterprise-Monolith — der Learning Skill skaliert mit der Komplexität der Codebase.
