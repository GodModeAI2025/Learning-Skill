Einen Kurs erstellen lassen

In fünf Schritten vom Code zum fertigen Kurs. Der Learning Skill führt Sie durch den Prozess — Sie müssen nur antworten.

01 Skill aufrufen 02 Zielgruppe wählen 03 Integrationsmodus 04 Der Skill arbeitet 05 Ergebnis nutzen
01

Skill aufrufen

Der Learning Skill wird direkt in Claude CodeAnthropics offizielle Kommandozeile für KI-gestützte Softwareentwicklung. Läuft im Terminal und hat Zugriff auf Ihr Dateisystem. aufgerufen. Navigieren Sie zuerst in das Projektverzeichnis, das Sie erklären lassen möchten:

# Terminal öffnen und ins Projektverzeichnis wechseln
$ cd /pfad/zu/meinem/projekt

# Claude Code starten
$ claude

# Den Learning Skill aufrufen
claude> /learning

Der Skill startet und erkennt automatisch die Codebase in Ihrem aktuellen Verzeichnis. Er stellt Ihnen dann einige Fragen, um den Kurs auf Ihre Bedürfnisse zuzuschneiden.

Sie müssen nichts installieren oder konfigurieren. Der Skill ist als Plugin direkt in Claude Code verfügbar, sobald er eingerichtet ist.

02

Zielgruppe wählen

Der Skill fragt, für wen der Kurs sein soll. Klicken Sie auf die Zielgruppen, die Sie brauchen — Mehrfachauswahl ist möglich:

👤

Anwender

Nutzen die Software, kennen keinen Code

💻

Entwickler

Arbeiten mit dem Code, brauchen technische Details

📊

Entscheider

Verantworten das Projekt, brauchen strategische Sicht

Wählen Sie oben eine oder mehrere Zielgruppen aus, um zu sehen, was generiert wird.

Je mehr Zielgruppen Sie wählen, desto mehr HTML-Dateien werden erzeugt. Jede Zielgruppe bekommt eigene Inhalte, Sprache und Tiefe.

03

Integrationsmodus

Der Learning Skill kann in zwei Modi arbeiten, je nachdem wie Sie die Kurse verwenden möchten:

📦

Standalone

Die Kurse werden als eigenständige HTML-Dateien in einem /kurs-Ordner erzeugt. Perfekt zum Teilen per E-Mail oder Fileserver. Jede Datei funktioniert unabhängig.

🔗

Eingebettet

Die Kurse werden so erzeugt, dass sie in bestehende Dokumentation oder Wikis eingebettet werden können. Enthält zusätzliche Metadaten und vereinfachtes Styling.

Empfehlung: Starten Sie mit Standalone. Das ist die Standardeinstellung und funktioniert in den meisten Fällen am besten.

04

Der Skill arbeitet

Nachdem Sie Zielgruppe und Modus gewählt haben, läuft alles automatisch. Der Skill durchläuft drei Phasen:

🔍

Analyse

Scannt alle Dateien, erkennt Struktur, Technologien, Abhängigkeiten und Muster.

📋

Curriculum

Erstellt einen Lehrplan: Welche Themen, in welcher Reihenfolge, auf welcher Tiefenebene?

⚙️

Build

Erzeugt die HTML-Dateien mit Inhalten, CSS, JavaScript, Navigation und Verlinkung.

Die Dauer hängt von der Projektgröße ab. Ein kleines Projekt (< 50 Dateien) ist in wenigen Minuten fertig. Große Projekte können 10-15 Minuten dauern.

05

Ergebnis nutzen

Nach Abschluss finden Sie die fertigen HTML-Dateien in Ihrem Projektverzeichnis. So nutzen Sie sie:

1

Dateien finden

Die Kurse liegen im /kurs-Ordner. Starten Sie mit der index_de.html (oder index_en.html).

2

Im Browser öffnen

Doppelklicken Sie auf die HTML-Datei oder ziehen Sie sie in Ihren Browser. Kein Server nötig.

3

Navigieren

Nutzen Sie die Breadcrumb-Navigation oben und die Sibling-Links unten, um zwischen Seiten zu wechseln. Deep-Dive-Links führen zu tieferen Ebenen.

4

Teilen

Senden Sie den kompletten /kurs-Ordner an Kollegen. Alle Links funktionieren relativ — kein Setup nötig.
